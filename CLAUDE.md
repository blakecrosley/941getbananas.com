# 941 Get Bananas

Part of the 941 ecosystem - product/marketing site.

## Stack
- FastAPI (Python)
- HTMX + Alpine.js + Bootstrap 5
- Jinja2 templates
- Railway deployment

## Commands
- `./run.sh` - Start development server
- `pip install -r requirements.txt` - Install dependencies

- `~/Projects/941return.com/` - Marketing site
